    各种数据库的默认端口（JAVA）

    Edited on 2022-07-04 In DB

    关系型数据库

    1、Oracle

    驱动：oracle.jdbc.driver.OracleDriver

    URL：jdbc:oracle:thin:@127.0.0.1:1521:dbname

    port：默认1521

    2、SQL Server

    驱动：com.microsoft.jdbc.sqlserver.SQLServerDriver

    URL：jdbc:microsoft:sqlserver://127.0.0.1:1433;DatabaseName=dbname

    port：默认1433

    3、MySQL

    驱动：org.gjt.mm.mysql.Driver

    URL：jdbc:mysql://127.0.0.1:3306/dbname

    port：默认3306

    4、postgres

    驱动：org.postgresql.Driver

    URL：jdbc:postgresql://127.0.0.1:5432/postgres?charSet=utf8&currentSchema=SchemaName

    port：默认5432

    5、pointbase

    驱动：com.pointbase.jdbc.jdbcUniversalDriver

    URL：jdbc:pointbase:server://127.0.0.1:9092/dbname

    port：默认9092

    6、DB2

    驱动：com.ibm.db2.jdbc.app.DB2Driver

    URL：jdbc:db2://127.0.0.1:5000/dbname

    port：默认5000

    NOSQL数据库（not only SQL）

    1. MongoDB port：默认27017
    2. Redis port：默认6379
    3. memcached port：默认11211
