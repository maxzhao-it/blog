    Mojo 工程

    Mojo 就是 Maven Plain Old Java Object。是对 Pojo 的一种改进。
    每一个 Mojo 都是 Maven 中的一个执行目标（executable goal），Maven plugin 就是对多个 Mojo 的封装。
