Java 使用 license

使用 truelicense

  1. 应用程序可以创建以及验证绑定给用户、系统等实体的 license。
  2. 防止用户随意拷贝软件和 license。
  3. licenses 可以是永久性的或者临时性的（在某个特定时期内有效）
  4. licenses 的验证由 JAVA Security API 提供的数字签名机制来实现。
  5. license 安装模块需要用特殊机制对其进行保护，以防被反编译轻易破解。
